# Universal-Domain-Adaptation-for-Cross-Building-Occupancy-Estimation-and-Activity-Recognition

**Official repository** of the "Universal Domain Adaptation for Cross-Building Occupancy
Estimation and Activity Recognition Under Sensor Heterogeneity"



## Overview

Smart buildings require models that generalise across different environments — but real-world sensor data varies between rooms, apartments and buildings (domain shift), and new activities appear that the model has never seen. **Universal Domain Adaptation (UniDA)** handles both challenges simultaneously without requiring any knowledge of which classes overlap between domains.

This repo adapts four state-of-the-art UniDA methods — originally designed for image data — to **one-dimensional smart-building sensor data**. All four methods share a common **Temporal Encoder** backbone and are evaluated on Occupancy Estimation and Activity Recognition
